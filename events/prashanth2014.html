Comprehensive Static Analysis of Embedded Software (C/C++, Ada) using Polyspace products

Speaker:Prashant M, Mathworks
Date: July 22, 2014

Abstract

The talk covers the following topics through illustrative examples.

  • Detect difficulty to find run time errors
  • Prove the software is safe
  • Reduce the Test and Verification Cycle
  • Track Software Quality Metrics
  • Enforce Code Rule Standards (MISRA C/C++, JSF )
